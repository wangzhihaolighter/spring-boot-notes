# 说明

spring-boot-notes 用于记录学习、使用 Spring Boot 框架。

主要内容是 Spring Boot 框架的一些特性、小功能的实现、常用框架的整合等。

Spring Boot版本：2.X

JDK版本：8

开发工具：IntelliJ IDEA
